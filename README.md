
# Flutter Application  

This project is built using the **Flutter framework** and integrates the following libraries to enhance functionality and user experience:  

- **`permission_handler`**: Manages permissions efficiently.  
- **`flutter_animate`**: Adds smooth and visually appealing animations.  
- **`flutter_native_splash`**: Creates a customizable splash screen.  
- **`GetX`**: Simplifies state management and navigation.  

Feel free to explore the **`lib`** folder to understand the file structure and source code. You are welcome to use the code in your own projects as needed!  

---

## Features  

- **Clean Architecture**: Modular and organized file structure for easy understanding and scalability.  
- **Dynamic Animations**: Interactive UI transitions using `flutter_animate`.  
- **Seamless State Management**: Powered by `GetX` for efficient app logic and navigation.  
- **Splash Screen**: A professional splash screen generated with `flutter_native_splash`.  

---

## Installation  

1. **Clone the Repository**  
   ```bash  
   git clone https://github.com/isikkerim/flutter-todo-app.git  
   cd flutter-todo-app  
   ```  

2. **Install Dependencies**  
   Fetch the required packages using:  
   ```bash  
   flutter pub get  
   ```  

3. **Run the Application**  
   Execute the app on your emulator or physical device:  
   ```bash  
   flutter run  
   ```
